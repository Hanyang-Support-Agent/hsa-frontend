# HSA 관리자 화면 레이아웃

각 화면의 텍스트 아트 와이어프레임이다. 사이드바는 모든 화면에서 동일하게 고정된다.

---

## 공통 사이드바

```
┌──────────────────────┐
│ HSA Admin            │
├──────────────────────┤
│ 대시보드             │
│ 문의 목록            │
│ 로그 조회            │
│ 문서 관리            │
├──────────────────────┤
│ [DEV] 문의 주입      │  ← 개발/테스트 전용
└──────────────────────┘
```
